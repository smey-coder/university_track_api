Here is a README.md document for your University Track Assignment Management Flow (Laravel API + React Admin + Teacher Role).

# 📚 Assignment Management Flow
## University Track System

Assignment Management allows Admin and Teacher users to create, update, view, and manage assignments for students.

The system uses **Role-Based Access Control (RBAC)** with Laravel Sanctum authentication and Spatie Permission.

---

# 1. User Roles

The Assignment module supports two main roles:

## 👑 Admin

Admin can:

- View all assignments
- Create assignments
- Select any teacher
- Select courses belonging to the selected teacher
- Update any assignment
- Delete assignments

Flow:


Admin Login
|
|
Assignment Management
|
|
Select Teacher
|
|
Load Teacher Courses
|
|
Create Assignment
|
|
Students can view assignment


---

## 👨‍🏫 Teacher

Teacher can:

- View own assignments
- Create assignments
- Update own assignments
- Select only courses assigned to themselves
- Cannot edit another teacher's assignment

Flow:


Teacher Login
|
|
System detects teacher profile
|
|
Load teacher courses
|
|
Create Assignment
|
|
Students receive assignment

# 3. Assignment Code Generation

Assignment code is generated automatically.

Format:


AS-{TeacherName}-{Year}-{Day}


Example:


AS-DARA-2026-MON


Structure:


AS
|
Assignment prefix

DARA
|
Teacher short name

2026
|
Current year

MON
|
Created day
